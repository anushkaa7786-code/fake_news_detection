What is the web application is about?
This application is about “Fake news detection” website which uses Machine Learning(ML) and Natural Language Processing(NLP) which is used to analyze news texts which is fake or real so that users can enter any news content and system can give good result or content to the whole world..

Why did you build it and what is the purpose of it?
We built the website/application to reduce the fake news that spreads around the social media or online platforms .
Users can’t able to find the news whether it is real or fake. The main purpose of this website/application to help the users to verify news or contents real or fake 
To improve more awareness about the social media or online platform. We provide a simple automated solution using machine learning(ML) instead of manual checking..

What motivated you to built the website?
We wanted to decrease the spread misinformation and fake news which is motivated me and my team members. This helps the society,politics and public’s opinion. This inspired our team members to learn machine learning(ML) and web development while we are creating a real world application or website..
How does your website differ from other fake news detection websites?
This is simple and user friendly interface . This gives quick result with minimum Inputs. Which is designed mainly for educational and academic purpose. Easy to understand and modify compare to complex systems..
